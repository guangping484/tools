📖 使用指南

奉天工具站 · 免费在线工具 · 无需注册

📝 文本工具箱

输入文字 → 点击「字数统计」「去重」「清理格式」→ 立即出结果

🖼️ 批量图片处理

选择图片 → 选格式（PNG/JPG/WebP）→ 点「转换」→ 点「下载」

🔐 密码管理器

设密码长度 → 点「生成密码」→ 点「复制密码」→ 粘贴使用

🎵 音频转换器

点击「在线转换」跳转外部工具，免费使用

📊 表格合并

点击「下载」安装 Mac App，导入 CSV/Excel 文件一键合并

📂 文件批量重命名

点击「下载」安装 Mac App，按规则批量重命名文件

📅 日期计算器

选开始日期 → 选结束日期 → 点「计算」→ 看相隔天数/月数/年数

🎲 随机抽奖

输入名单（每行一个）→ 设抽取人数 → 点「抽奖」→ 看中奖者

🔤 大小写转换

输入英文 → 选「全部大写」「全部小写」「首字母大写」「单词首字母大写」→ 看结果

📱 二维码生成器

输入文字或网址 → 点击「生成二维码」→ 点击「下载二维码」保存

🔐 Base64 编码/解码

输入文字 → 点击「编码」转成 Base64，或点击「解码」还原

📦 JSON 格式化

输入 JSON 数据 → 点击「格式化」美化排版，或「压缩」去除空格

📺 YouTube 缩略图下载器

输入 YouTube 链接 → 点击「获取缩略图」→ 选择分辨率下载

🔐 MD5 加密

输入文字 → 点击「生成 MD5」→ 获得 MD5 哈希值

💡 所有工具在本地浏览器运行，数据不上传
